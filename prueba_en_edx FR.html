Soutien au participant

Si vous ne trouvez les responses à vos questions ni dans les Questions fréquentes générales ni dans les Questions fréquentes techniques, complétez le formulaire que vous trouverez ci-dessous afin que nous puissions vous aider le plus vite possible.

Lorsque vous cliquerez sur Envoyer, une fenêtre pop-up vous avertira que le formulaire abien été enviyé et vous recevrez une notification par courriel avec votre numéro de demande. Merci de vérifier votre boîte spam.

Certificats
La Banque interaméricaine de développement propose tous ses cours en mode certificat vérifié et en mode assistant. Pour recevoir un certificat vérifié, il est nécessaire d'effectuer le paiement établi pour chaque cours et de le réussir avec un minimum de 65% avant la date limite établie dans le cours.

En mode assistant, le cours peut être suivi de manière totalement gratuite. Bien qu'aucun certificat ne soit délivré aux personnes inscrites dans ce mode, vous pourrez imprimer votre page de progression pour montrer que vous avez terminé le cours avec succès.

Dans le passé, edX proposait également des certificats "Honour Code". Toutefois, cette option n'existe plus. Si vous avez participé à un cours qui proposait un certificat "Honour Code", vous y aurez toujours accès dans votre tableau de bord edx.org.

Pour plus d'informations sur les certificats vérifiés, visitez la page des certificats vérifiés.

Si vous avez encore des questions, cliquez sur Générer un ticket de requête.

Registre des heures
La Banque interaméricaine de développement délivre un registre des heures qui devra être accompagné du certificat vérifié correspondant. Le registre des heures est disponible dans la section « Matériel du cours » dans la colonne de droite, ainsi que dans la page d'accueil du cours.

Tous les cours du programme IDBx de la Banque interaméricaine de développement peuvent être suivis gratuitement. Bien qu'il n'y ait pas de certificat pour ceux inscrits cours gratuits, vous pourrez imprimer votre page de progrès pour prouver que vous avez terminé le cours avec succès.

Si vous avez encore des questions, cliquez sur Créer un ticket de consultation

Paiements
La Banque interaméricaine de développement ne gère pas les paiements par le biais de la plateforme edX. Pour effectuer un paiement, vous devez vous rendre sur votre panneau de contrôle en cliquant sur votre nom, en haut à droite de la page edX, et cliquer sur le bouton « Opter pour un certificat vérifié » dans le cours correspondant. Une fois cela fait, vous pourrez payer avec une carte de crédit ou via PayPal. Si vous avez un code promo, n'oubliez pas de l'utiliser sur cet écran.

Si vous n’arrivez pas à effectuer un paiement, veuillez contacter info@edx.org.

Pour obtenir plus d'informations sur les certificats vérifiés, visitez la page des certificats vérifiés (disponible en anglais seulement).

Si vous avez encore des questions, cliquez surCréer un ticket de consultation.

NOTE : Si vous choisissez l’option « Autres problèmes », la gestion de votre demande nous prendra plus de temps.
